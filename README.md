# Работа с графикой

## Общие сведения
### Использованные технологии:
1) Многомодульность(1 app, 3 feature, 0 core, 0 data)
> так как нет взаимодействия с репозиториями типо Retrofit или Room, то нет data модулей. так же в силу изолированности фичей нет и core модулей
2) MVVM паттерн(Model View ViewModel)
> здесь стандартный паттерн: View - UI + UI логика, ViewModel - хранилище стейтов + бизнес логика + отдельный лайфсайкл с корутинами, Model - ее нет, поскольку нет Data модулей, а так в ней будет лежать DI
3) JetpackCompose - реактивный UI фреймворк
> Используется для создания UI через снапшоты и автоматическую работу с ними благодаря реактивности стейтов
4) Взаимодействие с системой андроид(Intent, activityResultRegistry)
> Необходимо для выбора изображения из галереи, а так же нужно для общения с другими активити/приложениями с помощью BroadcastReceiver и BindService
## Часть 1 - построение фрактала
> Фрактал - самоподобная математическая структура, применяющая рекурсивно несложные правила для самой себя

![изображение](https://github.com/user-attachments/assets/ac25d5c3-6787-4392-9342-4abfca6247c3)
### Функционал:
1) Возможность зумить и двигать картинку(очевидно качество становится с приближением хуже)
2) Возможность пересчета изображения в новых координатах, при этом зум и смещение обнуляются и появляется новая картинка
3) Рассчет картинки идет не в UI coroutine scope а в ViewModelScope, засчет чего можно свернуть приложение, и оно все равно досчитает

## Часть 2 - возможность рисования и закраски

![изображение](https://github.com/user-attachments/assets/b8660a87-ecc2-47e7-bc3c-4eb763b4276e)
### Функционал:
1) Возможность рисовать произвольные линии
2) Возможность рисовать прямые линии
3) Возможность рисовать окружность алгоритмом Брезенхама
4) Возможность рисовать кривые Безье итеративным алгоритмом
5) Возможность закраски модифицированным алгоритмом
6) Возможность закраски обычным алгоритмом
7) Возможность приближать и смещаться относительно картинки
8) Возможность вернуть действия назад или вперед(хранится целый стек картинок)
